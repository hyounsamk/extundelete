# extundelete
ext3 or ext4 undelete utility <br>

http://extundelete.sourceforge.net/

## Installing Dependencies

```sh
sudo apt install e2fsprogs ext2fs-dev
```

## Building source
```sh
./configure && make
sudo make install
```
